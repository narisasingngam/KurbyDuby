# Kurby Duby

A very cute Kurby collecting coins with joy and cheer.

## Prerequisite

- Python version 3.0 above.
----
## Installation

```
$ cd https://github.com/narisasingngam/KurbyDuby
```
----
## Run the application

```
$ python kurby.py
```
----
## Control
- Move left and right for moving player
